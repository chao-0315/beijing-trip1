7/3 (五)
D2		 故宮專屬日
天安門廣場 (搭車路過) → 人民大會堂 (搭車路過) → 故宮博物院 (上午精華遊，約下午二點出園) → 回酒店午休充電 → 景山公園 (下午四點後登頂看落日全景)[cite: 1] 		故宮餐廳或自備輕食
(故宮內部極大，在冰窖餐廳簡單用餐，爭取休息時間)[cite: 1]		 四季民福烤鴨店 (故宮店)
(看著故宮護城河夜景吃烤鴨，離牛街酒店開車僅 20 分鐘)[cite: 1]
長輩減負密訣：天安門、大會堂採車遊不下來排隊。約一、二點回酒店睡午覺，下午四點後天氣轉涼再出門爬景山 (約走15分鐘石階)。[cite: 1]
7/4 (六)
D3		 長城專屬日
慕田峪長城 (含全包式箱式纜車往返) → 下山專車返程 → 鳥巢 / 水立方 (外觀絕美夜景)[cite: 1] 		慕田峪長城小鎮餐廳
(長城腳下享用精緻農家菜，補充電力)[cite: 1]		 局氣 (奧運村店)
(創意北京菜，欣賞完鳥巢水立方夜景後順路用餐)[cite: 1]
長輩防暑提醒：7月郊區暴曬。務必多帶水水、補充電解質。早上 07:30 前出發避開出城車潮。長城上拍照吹風即可，切勿讓長輩強爬陡峭城牆。[cite: 1]
7/5 (日)
D4		 十三陵與淘寶日
明十三陵 (定陵地宮或長陵大殿) → 古玩物地攤 (潘家園 或 報國寺) → 全程專車接回酒店 (放棄搭乘地鐵)[cite: 1] 		昌平昌盛源春餅 或 景區農家院
(體驗昌平當地的春餅宴或特色烤鱒魚)[cite: 1]		 牛街聚寶源 (總店)
(牛街神級名店！走路即到。體驗最正宗的炭火銅鍋涮羊肉與手切鮮羊肉)[cite: 1]
週末地攤提醒：潘家園地攤在週六、週日最為熱鬧，今天去剛好可以體驗最完整的古玩集市。長輩買好戰利品後，直接由專車接回酒店。下午 4:30 記得在車上先線上用 APP 點評抽取聚寶源大桌號碼牌。[cite: 1]
7/6 (一)
D5		 西北皇家園林線
頤和園 (精華遊船路線) → 圓明園 → 北京大學 (外觀西門拍照，不進校園)[cite: 1] 		白家大院
(開在海淀區，清宮宮廷菜，服務員身著旗袍，園林環境極美)[cite: 1]		 小大董 (海淀中關村店)
(精緻現代北京菜，適合看完北大後順路前往)[cite: 1]
頤和園精華遊船路線指南：東宮門入園→仁壽殿→樂壽堂→文昌閣碼頭上船乘大遊船賞昆明湖、十七孔橋、佛香閣→石舫碼頭下船→直接進入全長728公尺的彩繪長廊往東散步（有遮蔭、有木椅）→走到長廊中段的排雲門附近→向北穿過無障礙通道→走向北宮門出園。千萬不要去爬「萬壽山佛香閣」。北大因缺乏遮蔭且純步行，門口拍照留念即可。[cite: 1]
7/7 (二)
D6		 南城歷史與步行街
正陽門城樓 → 中國國家博物館 (上午冷氣參觀) → 中午烤鴨 → 下午回酒店睡午覺 → 天壇公園 (傍晚祈年殿散步) → 大柵欄 / 王府井步行街 (有時間再逛)[cite: 1] 		便宜坊烤鴨 (前門總店)
(前門老字號悶爐烤鴨，國博參觀完後專車送去10分鐘即到)[cite: 1]		 王府井百貨地下「和平菓局」
(沉浸式復古老北京街區，內部可品嚐各種特色老胡同小吃)[cite: 1]
國博內部有冷氣、電梯，非常平坦舒服。中午吃飽後必須嚴格執行下午回酒店午休。傍晚 4:15 天氣轉涼再前往天壇 (天壇與前門離牛街極近，專車往返極方便)。[cite: 1]
7/8 (三)
D7		 老北京王府與胡同
恭王府 (半天慢走) → 北海公園 → 什剎海 / 煙袋斜街 (體驗胡同三輪車)[cite: 1] 		烤肉季 (什剎海總店)
(百年老字號，主打「烤羊肉」，位於銀錠橋旁，風景極佳)[cite: 1]		 梅府家宴 或 胡同私房菜
(什剎海胡同內的四合院私房菜，環境儒雅，體驗老北京王府宅門風情)[cite: 1]
🚨 備忘提醒：今日 12:30 起開放 7/10 回程 中華航空 網路報到與預選座位，請點此連結準時劃位！
恭王府面積適中，適合長輩。下午安排全團搭乘「老北京胡同三輪車」遊什剎海，長輩不用動腿，晚上在南鑼鼓巷南口上專車回酒店。[cite: 1]
7/9 (四)
D8		 東城文青與現代
雍和宮 (燒香祈福) → 孔廟/國子監 (古樹散步) → 五道營胡同 → 三里屯太古里 (青年組行程，長輩可彈性先回酒店)[cite: 1] 		京兆尹 (雍和宮店) 或 五道營特色西餐
(京兆尹為米其林三星素食，環境充滿禪意，適合文化之旅)[cite: 1]		 新榮記 (三里屯店) 或 太古里潮流餐廳
(享用精緻美食，為這次北京之行畫下完美的現代時尚句點)[cite: 1]
雍和宮、五道營為同一區域，步行即可。長輩累了可由專車先載回牛街酒店午休，年輕人可留在三里屯太古里逛街，傍晚專車再接長輩來三里屯會合吃晚餐。[cite: 1]
